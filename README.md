# Digital Clock ⏰

A simple and elegant Digital Clock built using HTML, CSS, and JavaScript. The clock displays the current hours, minutes, and seconds in real-time and updates automatically every second.

## 🚀 Features

- Real-time clock updates
- Displays Hours, Minutes, and Seconds
- Clean and modern user interface
- Lightweight and beginner-friendly project
- Built with pure HTML, CSS, and JavaScript

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

├── index.html
├── style.css
└── README.md

## ⚙️ How It Works

The project uses JavaScript's `Date()` object to fetch the current time and updates the clock every second using `setInterval()`.

## 🎯 Learning Objectives

This project helps beginners understand:

- DOM Manipulation
- JavaScript Date Object
- setInterval Function
- Real-Time UI Updates
- Front-End Development Basics

## 📸 Preview

A modern digital clock displaying the current time in HH:MM:SS format.
